# AI Football Quiz Generator

Final project for the Building AI course

## Summary

AI Football Quiz Generator is an AI-powered application that creates football quizzes based on the FIFA World Cup. It helps football fans, students, and quiz enthusiasts test their knowledge by generating multiple-choice and true/false questions instantly.

## Background

Football is the world's most popular sport, and millions of fans enjoy testing their knowledge of the FIFA World Cup. However, creating new quiz questions manually takes time. This project uses AI to automatically generate football quizzes from World Cup facts and statistics.

Problems this project solves:
- Saves time by generating quiz questions automatically.
- Makes learning football history more fun.
- Provides unlimited quiz practice.
- Encourages fans to learn more about the FIFA World Cup.

My motivation is that I enjoy football and wanted to create a simple AI project that combines sports with learning.

## How is it used?

The user opens the application and selects a quiz topic, such as:
- FIFA World Cup Winners
- Famous Players
- Records and Statistics
- World Cup History

The AI then generates a set of questions with multiple-choice answers. At the end of the quiz, the user receives a score and the correct answers.

The application is useful for:
- Football fans
- Students
- Teachers
- Sports quiz competitions

Example:
1. Select a topic.
2. Choose the number of questions.
3. Answer the quiz.
4. View the final score and correct answers.

## Challenges

The AI may occasionally generate incorrect or repeated questions. It also depends on accurate and up-to-date football data. The application should verify facts before presenting them to users.
